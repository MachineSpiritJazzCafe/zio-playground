# ZIO Playground

Playing with ZIO 2.x


## Setup

- **Scala:** 3.3.4
- **ZIO:** 2.1.14
- **Build Tool:** sbt 1.11.7

## Running
```bash
# Compile
sbt compile

# Run main
sbt run

# Tests
sbt test

# REPL
sbt console
```

## Structure

Topics for exploring:
- ZIO fundamentals & effect system
- Error handling with typed errors
- Resource management (Scope, ZLayer)
- Concurrency (Fibers, Ref, Queue)
- ZIO Test
- and more...
